# React Native Collapsible Page Header With Scroll Indicator

Collapsible Page Header With Scroll Indicator in a React Native App

# Getting Started

## Install modules:

```npm install```

### Install for iOS:

```
cd ios
pod install
```

## Start

```bash
# using npm
npm start

# OR using Yarn
yarn start
```
